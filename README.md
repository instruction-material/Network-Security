# Network Security

Local starter and solution materials for the `Network Security` course.

## Projects

- `NSEC1-Local-Port-Inventory-Tool`: classify local listeners by bind scope and exposure risk
- `NSEC2-Request-Schema-Validation-Gateway`: validate request shape and boundary rules before deeper logic
- `NSEC3-Suspicious-Request-Log-Parser`: parse request logs and summarize suspicious patterns
- `NSEC4-Rate-Limit-and-Abuse-Detector`: group traffic by actor and recommend throttle or block actions
- `NSEC5-Secure-Real-Time-Notifier`: validate connection state and real-time events for a small notification system
- `NSEC6-TLS-and-Proxy-Companion-App`: audit proxy and TLS configuration choices for exposed services

## Notes

- The pack uses `TypeScript` throughout to match the catalog course design.
- The examples stay defensive and lab-scoped; nothing here is intended for unsolicited scanning or offensive use.
- Each project folder contains a `starter/` and `solution/` implementation plus a short lab README.
